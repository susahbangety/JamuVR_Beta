Untuk Drawer
Pintu drawer namanya "right_cabinet_drawer_door tkz"
Daleman drawernya pake "Right Cabinet_Right txr"
Handle/pegangannya pake "handle_drawer txr"
kedua texture itu dipake buat ke semua meja yang ada drawernya (cooking table, ingredient table)

Untuk pintu di meja
Pintunya pake "right_cabinet_door txr"
Handle/pegangannya pake "handle_drawer txr" (iya sama kaya pegangannya drawer)
Texture itu dipake buat ke semua pintu meja (cooking table, ingredient table)

Untuk Meja tengah
Pake texture "Middle Cabinet txr"

Untuk Meja Kiri
Meja kiri bagian bawah (yang kotak doang) pake texture "Left Cabinet txr"
Untuk bagian yang kepisah (cabinet_back/cabinet_left dll) sesuain sama texturenya aja namanya

Untuk Meja Kanan
Untuk semuanya (Right Cabinet_Top, back, right, left, bottom tatakan) pake texture "Right Cabinet_Right txr"

Untuk Meja paling atas yang nama objeknya "Cookiing Table"
Ini pake texture "Bottom txr"
